Submission

When you have completed all three exercises, you will need to submit your work to Canvas.

Here is the step-by-step submission process:

  1. Zip your lab folder. The folder should contain the following files:
    • multiply.py
    • KmToMiles.py
    • MilesToKm.py
  2. Submit the zipped file (Lab01.zip) to Canvas.

 

 

Outline

Before submitting your assignment, step through this checklist to be certain that everything is complete. You should also check the grading rubric to see if there is anything that you might have forgotten.

Here is the step-by-step outline

  1. Create the Lab folder structure
  2. Read (and understand) the pseudocode as described in the lab instructions
  3. Create a new .py file and write the code for the sample exercise, run it and view the result
  4. Create a new KmToMiles.py file and write the code for your solution (follow the example and modify as needed from step 4). Run it and check your results.
  5. Create a new MilesToKm.py file and write the new code, test, and confirm the result.
  6. Zip your files and submit to Canvas (must be a zip file).

 

 

Hint: your python code files must have the following on the first line: 

#!/usr/bin/env python3

The hash character ('#') must be the first character of the .py source file.
