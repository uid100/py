Submission

When you have completed the lab exercises, you will submit your work to Canvas.

As before, include all your python source files and the requested test data text (.txt) files. Zip your lab folder and submit the Lab02.zip file to Canvas.

Reminders

  • Be sure to test all your code and make sure your calculations are correct.
  • Make sure your code meets the requirements specified in each exercise.
  • All your identifiers, including filenames, variables, and constants, should have clear, meaningful, and descriptive names.
    • "exercise1" does not describe the file or its contents.
    • "value1" could mean anything!
    • "height_in_inches" clearly describes a variable representing a height measured in inches. Anyone reading the code would understand its purpose.
    • "MONTHS_PER_YEAR" clearly describes a constant representing the number of months in a year. Anywhere in the code, anyone would understand its purpose.
  • Do not forget the required comments at the beginning of each source code file, including:
    • Your name
    • Course name and number
    • Lab number and exercise number
    • Brief description of the program
  • Include a table of test data and expected results for each exercise.
